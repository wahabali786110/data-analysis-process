# 📊 Data Analysis Process

This repository documents my systematic approach to the Data Analysis lifecycle. It contains Python scripts and Jupyter Notebooks detailing how to extract raw data from the wild, structure it, clean it, and perform statistical analysis before feeding it into Machine Learning models.

## Tech Stack
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Data Gathering:** Requests, BeautifulSoup (Web Scraping), APIs
* **Visualization:** Plotly Express, Matplotlib

---

## Learning Roadmap & Table of Contents

### Phase 1: Data Gathering
* [ ] **API Extraction:** Fetching live data from REST APIs using the `requests` library.
* [ ] **Web Scraping:** Extracting tabular data from HTML pages.
* [ ] **Database Connections:** Writing SQL queries to pull data into Pandas DataFrames.

### Phase 2: Data Assessing & Cleaning
* [ ] **Handling Missing Data:** Strategies for dropping or imputing `NaN` values.
* [ ] **Data Type Conversion:** Ensuring clean numerical and categorical columns.
* [ ] **Outlier Detection:** Identifying anomalies in raw datasets.
* [ ] **Standardization:** Fixing corrupted text strings and formatting issues.

### Phase 3: Applied Statistics & EDA (Exploratory Data Analysis)
* [ ] **Univariate Analysis:** Understanding distributions using histograms and boxplots.
* [ ] **Bivariate Analysis:** Finding correlations using scatter plots and regression lines.
* [ ] **Multivariate Analysis:** Exploring high-dimensional data using interactive pairplots and hue mapping.
